# Daily_App

2019年1月26日から毎日1日1個クソアプリを作っていきます。

ノリでツイートしてこれが事の発端。

# ルール

- 毎日21時に作ったアプリの内容をツイートする
- ストックはありとする
- もし遅れた場合は次の日に2つ作る
